Some notes about HTML Tags

When writing html, we should always keep in mind two types of tags: Paired and Singular.

Examples of paired tags

The Paragraph tag
We use this tag when there is a block of text that we want to define as a paragraph. Having the text between the opening tag and the closing tag will systematically add a space before and after the content of the paragraph.
The Emphasis tag
If you ever have words in your text that you want to slightly emphasize compared to the rest, this would be the tag to use.
The Html tag
Probably the most important tag of all. It is mandatory that all the content added to the page is inside the opening and the closing html tags.

Examples of singular tags

The Line break tag
This tag is used whenever it is important to create a division in lines. It should never be used as a design tool in order to create more space.
The Horizontal line tag
It is used when there is a change of topic in the content. It creates a smooth transition from one aspect of the content to the other.
